C++ is classified as a general purpose, object-oriented programming language. It is an extension of the original "C"
programming language. It is an intermediate(or mid level) language, as it contains high level and low level language features.
When first created, it was called "C with classes" because it implemented properties with the concept of "classes", but was renamed to
C++ in 1983. It is a very popular language for system or application software, drivers, and embedded software. It is favored for its
collection of predefined classes, and the ability to create user-defined classes.
