Python is considered an interpreted, object-oriented high-level language. It is preferred by those in Rapid Application Development. Python is a great language for beginners because of its simple syntax and readability. Python is dynamic, which means that data types do not have to be specifically declared when creating a variable. Many programmers love Python for its increased productivity, and the lack of a compilation step.

Example of dynamic variable: s = "This is a string"

Though no datatype was offically stated, Python knows that s is a string based on its content.
