So what are programming langauges? Well, every line of code is read like an instruction for the computer to do something.
Programming languages can be divided into the following categories:
  • Interpreted Programming Languages
  • Functional Programming Languages
  • Compiled Programming Languages
  • Procedural Programming Languages
  • Scripted Programming Languages
  • Markup Programming Languages
  • Logic-based Programming Languages
  • Concurrent Programming Languages
  • Object-oriented Programming Languages
Low vs. High level Programming: Low level languages are like machine code in that it is very specific and harder to understand.
