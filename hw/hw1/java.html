Java is a general-purpose, concurrent, object-oriented, class-based programming langauge that is very similar to C++. Java
includes a JRE, or a Java Runtime Environment, and the JDK, also known as the Java Development Kit. The Runtime Environment
provides all the class libraries and support files during runtime for the JVM. The JDK is used to compile, document, and package Java
programs. Java can be considered an easier language for the reason that it removes complexities like pointers and operator overloading
unlike other languages.
